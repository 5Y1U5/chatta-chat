# chatta-chat 事業開始チェックリスト

## 必須（リリース前）

- [ ] 電気通信事業の届出（総務省・管轄の総合通信局）
  - 費用: 無料
  - 必要書類: 届出書（様式第8号）、サービス説明書（様式第4号）、ネットワーク構成図（様式第3号）、登記事項証明書
  - 参考: https://www.soumu.go.jp/menu_seisaku/ictseisaku/denkitsushin_suishin/tetsuzuki/kokunaihoujin.html
- [ ] 利用規約の作成・公開
- [ ] プライバシーポリシーの作成・公開
- [ ] 特定商取引法に基づく表記の作成・公開（有料サービス提供時）

## 必須（有料プラン提供開始時）

- [ ] 決済機能の導入（Stripe Checkout + Webhooks + Customer Portal）
- [ ] 料金プラン設計の確定
  - フリー: 無料（5ユーザー、AI月50回制限）
  - スタンダード: 980円/ユーザー/月（年払い）、1,180円（月払い）
  - プロ: 1,680円/ユーザー/月（年払い）、1,980円（月払い）
- [ ] プラン制限の実装（ユーザー数上限、AI回数制限等）

## 推奨（信頼獲得・マーケティング）

- [ ] LP（ランディングページ）の作成
- [ ] セキュリティページの作成（データ保管、暗号化、インフラ情報）
- [ ] 独自ドメインの取得・設定
- [ ] ロゴ・ブランディングの整備

## 将来的（売上拡大後）

- [ ] プライバシーマーク取得（費用: 50〜100万円）
- [ ] ISMS（ISO27001）認証取得（費用: 100〜300万円）
- [ ] SOC2 Type II 取得（費用: 300万円〜）

## セキュリティ訴求ポイント（LP・セキュリティページ用）

| 項目 | 現状 | 訴求文 |
|------|------|--------|
| 通信暗号化 | Vercel（HTTPS/TLS標準） | 全通信をTLS 1.3で暗号化 |
| データ保管 | Supabase Pro（AWS東京リージョン） | データは日本国内のサーバーに保管 |
| DB暗号化 | Supabase（保存時暗号化） | 保存データはAES-256で暗号化 |
| 認証 | Supabase Auth | パスワードはハッシュ化して保存 |
| アクセス制御 | ワークスペース単位の分離 | テナント間のデータは完全に分離 |
| バックアップ | Supabase Pro（日次自動） | 毎日自動バックアップ、7日間保持 |
| インフラ | Vercel + Supabase | SOC2取得済みの信頼性の高いインフラを採用 |
